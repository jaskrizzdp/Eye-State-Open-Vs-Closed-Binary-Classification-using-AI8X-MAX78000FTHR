# Eye State Detection (Demo for Drowsiness Alert)

**Binary classification of eye states (open vs. closed) using AI8X on MAX78000FTHR. Demonstrates potential for drowsiness detection applications.**

---

## Project Overview

This project implements a **proof-of-concept eye state detection system** that classifies whether a person's eyes are open or closed. It showcases how such a system could be used for **drowsiness detection**, improving safety in scenarios like driving or operating machinery.  

By using the **AI8X platform**, the model is trained and optimized for **edge deployment** on the **MAX78000FTHR** microcontroller. The setup demonstrates fast, low-power, real-time inference on embedded hardware, highlighting the potential for safety-focused applications.

---

## Key Features

- **Real-time eye state detection**: Open vs. closed classification  
- **Binary classification AI model** optimized for microcontroller deployment  
- **Edge AI deployment** using MAX78000FTHR and onboard camera  
- **Low-power, high-speed inference** using the AI8X platform  
- **Demonstrates potential for drowsiness monitoring**  

---

## Applications

- Concept demonstration for driver drowsiness alert systems  
- Industrial operator safety monitoring proof-of-concept  
- Wearable or portable edge AI safety devices  

---

## How It Works

1. **Camera Input**: MAX78000FTHR captures real-time images of the eyes.  
2. **Preprocessing**: Input images are normalized and resized to fit the trained model.  
3. **AI Inference**: The AI8X-optimized binary classification model predicts whether the eyes are open or closed.  

---

## Hardware & Software

- **Microcontroller**: MAX78000FTHR  
- **Platform**: AI8X Edge AI  
- **Camera**: Onboard MAX78000FTHR camera  
- **Model Type**: Binary classification (Open vs. Closed Eyes)
